# ChatFlow 💬

A WhatsApp-like web chat app built with Flask + SQLite.

## Deploy to Render (free)

See the step-by-step guide below, or click the button if you have render.yaml set up.

## Run locally

```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:5000
```
